A powerful command-line interface (CLI) tool for AI-powered text processing. This toolkit allows users to summarize text, translate languages, and analyze sentiment directly from the terminal with beautiful formatted output.

---

# ✨ Features

- 📝 AI Text Summarization
- 🌐 Multi-language Translation
- 😊 Sentiment Analysis
- 🎨 Beautiful Terminal UI
- 📁 File Input & Output Support
- 🔄 Interactive CLI Mode
- 📊 Token Usage Tracking
- 🎭 Demo Mode Support
- 📄 JSON & Markdown Output
- ⚡ Fast and Lightweight
