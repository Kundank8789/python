A powerful command-line interface (CLI) tool for text processing using AI. Summarize text, translate between languages, and analyze sentiment with beautiful terminal output.

## ✨ Features

- 📝 **Text Summarization** - Condense long text into short, medium, or long summaries
- 🌐 **Multi-language Translation** - Translate between 10+ languages including Spanish, French, Japanese, and Chinese
- 😊 **Sentiment Analysis** - Detect emotions, confidence scores, and intensity levels
- 🎨 **Beautiful Output** - Rich formatting with colors, panels, and progress indicators
- 💾 **File I/O Support** - Process text from files, direct input, or stdin
- 🔄 **Interactive Mode** - REPL-style interface for rapid testing
- 📊 **Token Tracking** - Monitor API usage and costs
- 🎭 **Demo Mode** - Works without API credits for testing and development
- 📁 **Multiple Output Formats** - Text, JSON, or Markdown
